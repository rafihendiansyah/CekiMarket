# Folder Banner Promo

## Cara Menggunakan

1. **Simpan gambar promo di folder ini** dengan nama:

    - `promo1.jpg` (atau `.png`, `.webp`, `.jpeg`)
    - `promo2.jpg`
    - `promo3.jpg`
    - dst... (maksimal promo10)

2. **Format yang didukung:**

    - JPG / JPEG
    - PNG
    - WebP

3. **Ukuran gambar yang disarankan:**

    - Lebar: 1200px - 1920px
    - Tinggi: 300px - 600px
    - Rasio: 16:9 atau 4:1 (landscape)

4. **Cara kerja:**

    - Sistem akan otomatis membaca semua file `promo1.jpg`, `promo2.jpg`, dll
    - Gambar akan muncul di carousel di halaman home
    - Carousel akan slide otomatis setiap 5 detik

5. **Contoh nama file:**
    ```
    promo1.jpg  ✅ Akan muncul
    promo2.jpg  ✅ Akan muncul
    promo3.png  ✅ Akan muncul
    banner1.jpg ❌ Tidak akan muncul (harus promo1, promo2, dll)
    ```

## Tips

-   Kompres gambar agar tidak terlalu besar (max 500KB per gambar)
-   Gunakan nama file yang konsisten: promo1, promo2, promo3, dll
-   Jika ingin mengubah urutan, ubah nomor file (promo1 akan muncul pertama)
